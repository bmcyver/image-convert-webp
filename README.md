# Obsidian Image To WebP

개인 목적으로 개발되었으며, [Image Converter](https://github.com/xRyul/obsidian-image-converter)을 상당부분 참고하여 제작하였음.

- 지원되는 입력 이미지: PNG, JPG, WebP, AVIF, HEIC, HEIF
- 지원되는 출력 이미지: WEBP/85 (단, AVIF는 처리하지 않음)
- 이미지 저장 경로: `assets/{{YEAR}}/{{FILE_BASE_NAME}}-{{TIMESTAMP}}.webp`
- 포함된 라이브러리: [heic-decode](https://github.com/catdad-experiments/heic-decode)
- 번들 사이즈: ~1.4MB
